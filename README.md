## 📚 正在构建的 CS 底座 (Core CS Syllabus Under Construction)
*我以自律的系统化方式攻克计算机科学核心，并在 Obsidian 知识库中完成深度沉淀，每一门课都是我构建技术底座的基石：*

*   📐 **离散数学 (Discrete Math)** — 为形式化逻辑推理、图论建模与空间计算奠定数学底座。
*   💾 **数据结构与算法 (DSA)** — 优化高吞吐量数据流与空间计算的内存开销与时空效率。
*   🖥️ **系统原理与微架构 (OS & Architecture)** — 理解进程调度、多线程并发、虚拟内存与硬件屏障，赋能高性能计算。
*   🌐 **网络协议与通信安全 (Networking & Security)** — 构建低丢包、高实时性的端到端分布式传感器/媒体数据流传输。
*   🏗️ **软件工程方法论 (Software Engineering)** — 实践模块化架构设计、规范化版本控制与敏捷开发流程。
