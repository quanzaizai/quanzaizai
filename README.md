## 👤 关于我 (About Me)

> 💡 **探索者申明 (My Declaration)**：

- **工程主攻 (Focus)**：目前主攻**边缘端低延迟通信**与**多模态空间感知**，研究如何利用有限的边缘算力解算复杂的三维物理世界数据。

*我以自律的系统化方式攻克计算机科学核心，并在 Obsidian 知识库中完成深度沉淀，每一门课都是为物理 AI 构建的基石：*

*   🏗️ **软件工程方法论 (Software Engineering)** — 实践模块化架构设计、规范化版本控制与敏捷 CI/CD 流程。
