### 👨‍💻 关于我 (About)

💡 **探索者申明**：作为一名坚信**“做中学 (Learning by Doing)”**的自学者，我选择以**“公开学习/研发 (Build in Public)”**的形式，将自学路径与知识库的演进透明化。

- 🎯 **愿景定位**：打牢计算机系统支撑，探索其与智能算法的结合。不预设死分方向，通过硬核的工程能力连接代码世界与实际应用。
- ⚒️ **研发哲学**：无论是系统简单的运算调度，还是算法的高效推理，都通过“优先动手实现”来夯实底层理解。
- 📚 **当前焦点**：沉淀「数据结构与算法」、「网络协议」与「操作系统」基础底座，同时关注机器学习工程化落地。
